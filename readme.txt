☆ MasicGirl ☆
魔法少女を操作して敵を打ち倒すシューティングゲームです。

===== 遊び方 =====
1.タイトル　─　2.ステージ選択画面　┬　3.シューティング　┬　5.ゲームオーバー
                                    └　4.成長画面        └　6.ゲームクリア


###　1.タイトル
タイトル画面です。マウスをクリックするとステージ選択画面へ移ります。


###　2.ステージ選択画面
ステージを選択できます。STAGE1　<　STAGE2　<　STAGE3　の順で難しくなりますが、どのステージからでも挑戦できます。
左上のプレビューウィンドウで「3.シューティング」での魔法少女の挙動を確認できます。
右下の「成長」をクリックすると「4.成長画面」へ移ります。


###　3.シューティング
操作説明　┬　「WASD」で魔法少女を移動できます。
　　　　　├　「スペース」で攻撃できます。
　　　　　└　「E」でいつでも中断できます。

左上の「SCORE」が現在のあなたのスコアです。敵を倒すことで上昇します。

中央上の「MATERIAL」がステージ内で入手したマテリアルです。「4.成長画面」で魔法少女のステータスを上昇させるために使用します。
マテリアルは、各ステージに「緑」・「赤」・「青」の3種類がひとつずつ出現します。

左下の「♡」が魔法少女の「生命力」です。敵から攻撃を受けると「♡」がひとつ減少します。追加で一定時間魔法少女が無敵になりますが、攻撃できなくなります。
「♡」がゼロになると「5.ゲームオーバー」になります。

ステージ最後に現れる青色の「クリスタル」を入手することで「6.ゲームクリア」となります。


###　4.育成画面
「3.シューティング」で入手したマテリアルを使用して魔法少女のステータスを上昇させることができます。
ステータスを上昇させることでゲームを有利に進めることができます。

ステータス　┬　生命力：「3.シューティング」での「♡」の個数です。「緑」のマテリアルを使用することで成長します。
　　　　　　├　魔法力：「3.シューティング」で敵に与えるダメージです。「赤」のマテリアルを使用することで成長します。
　　　　　　└　機動力：「3.シューティング」での魔法少女の移動速度です。「青」のマテリアルを使用することで成長します。

ステータス右のボタンをクリックすると成長させることができます。成長可能なときはボタンが赤色になります。
ステータスを成長させるごとに、次に成長させるために必要なマテリアル数が増加します。


###　5.ゲームオーバー
再挑戦するか、「2.ステージ選択画面」に戻るか選択できます。


###　6.ゲームクリア
ステージでの「SCORE」と「MATERIAL」とランクが表示されます。
ランクは「SCORE」に応じて変動します。 例)SCOREが220000 --> ランクA
「3.シューティング」で集めたマテリアルを使用して「4.育成画面」で魔法少女を成長させることで「SCORE」を稼ぎやすくなります。
全ステージランクSを目指してみてください。
